# Distrobox

This directory contains useful scripts for setting up development environment in a [distrobox](https://github.com/89luca89/distrobox) container for **Errands**.

- [setup-distrobox.sh](setup-distrobox.sh) Script to setup **Arch Linux** distrobox with all of the development libraries, [Zed Editor](https://github.com/zed-industries/zed) and Neovim
